# 🚀 Personal AI Employee – Hackathon Project

## 📌 Overview
This project is an AI-powered autonomous employee system that automates business workflows such as email management, LinkedIn posting, Facebook page automation, and Odoo accounting integration.

It works as a self-running AI agent system — just execute the project and it handles tasks automatically without manual intervention.

---

## ⚙️ Features

### 📧 Gmail Automation
- Reads and processes incoming emails  
- Categorizes emails into Inbox, Needs_Action, and Done  
- Auto-response generation using AI  

---

### 💼 LinkedIn Automation
- Generates AI-based business posts  
- Automatically posts content on LinkedIn  
- Helps in lead generation and business growth  

---

### 📘 Facebook Page Automation
- Automatically opens Facebook page workflow  
- Creates and posts content on business page  
- Stores full posting history  

---

### 🧾 Odoo Accounting Integration (Docker)
- Self-hosted Odoo system using Docker  
- AI-generated accounting scripts  
- Stores financial records automatically  
- Generates weekly CEO reports  

---

### 📊 CEO Weekly Report System
- Automatically generates `CEO_WEEKLY.md`  
- Summarizes business performance  
- Saves reports for review and tracking  

---

### 🔁 Fully Autonomous System
- Runs without manual input  
- AI agent handles multi-step workflows  
- Continuous automation loop for business tasks

5. System starts automation automatically
