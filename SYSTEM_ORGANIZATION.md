# Odoo Invoice Monitoring System - File Organization Guide

## Core System Files (These are the files you need for the system to work):

### Main Execution Files:
- `sync_odoo_invoices_to_ceo_report.py` - Main script that syncs Odoo invoices to ledger and generates CEO reports
- `monitor_odoo_invoices.py` - Continuous monitoring script that periodically checks for new invoices
- `run_invoice_monitor.bat` - Menu-driven batch file to run the system easily

### Core Skills (Already in your system):
- `Skills/Odoo_Integration/odoo_integration.py` - Connects to Odoo and fetches invoice data
- `Skills/Ledger_Manager/ledger_manager.py` - Manages local financial records
- `Skills/CEO_Briefing_Generator/ceo_briefing_generator.py` - Generates CEO reports
- `Skills/Audit_Logger/audit_logger.py` - Logs system activities

### Generated Reports:
- `Reports/CEO_Reports/CEO_Weekly_Briefing_YYYYMMDD.md` - Weekly CEO reports generated by the system

## Configuration Files:
- `Skills/Odoo_Integration/config.json` - Odoo connection settings (URL, database, credentials)
- `Skills/Ledger_Manager/config.json` - Ledger database settings
- `Skills/CEO_Briefing_Generator/config.json` - Report generation settings

## Supporting Files (Used by the system but you don't need to interact with them directly):
- All other files in the Skills folders
- Various utility scripts and configuration files

## Files You Can Safely Ignore/Don't Need to Modify:
- Most files in the root directory that don't start with "sync_" or "monitor_"
- Test files and backup files
- MCP server files (unless you need advanced integration)

## How to Use the System:
1. Start with: `run_invoice_monitor.bat` (double-click this file)
2. Choose "Option 1" for one-time sync
3. Check reports in: `Reports/CEO_Reports/` folder
4. For continuous monitoring, run: `monitor_odoo_invoices.py`

## Key Points:
- The system is already connected to your Odoo instance (localhost:8069)
- New invoices are automatically fetched and added to your financial records
- CEO reports are generated automatically with the latest invoice data
- All reports are saved in the Reports/CEO_Reports folder
